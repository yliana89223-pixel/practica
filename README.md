# Проект по практике "Учет самокатов около метро"
Данный проект разработан в рамках производственной практики по направлению **09.03.01 «Информатика и вычислительная техника»**.

**Цель** — создать инструмент, который помогает городским службам и аналитикам отслеживать загруженность парковок электросамокатов у станций метро. Пользователь загружает фото, а нейросеть определяет и подсчитывает все самокаты на изображении, отображая результат с визуальными рамками.

## Используемые технологии

- **Язык**: Python
- **Фреймворк**: Flask 
- **Нейросеть**: YOLOv8 - дообучена на собственных изображениях
- **Обработка изображений**: OpenCV, NumPy
- **База данных**: SQLite
- **Фронтенд**: HTML, CSS, JavaScript

## Функциональность

- Загрузка изображений через веб-интерфейс
- Автоматическое обнаружение и подсчет самокатов
- Визуализация результатов (рамки вокруг объектов)
- Сохранение истории запросов в БД (дата, имя файла, количество объектов)
- Просмотр истории обработанных изображений
